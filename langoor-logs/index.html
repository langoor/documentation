Langoor Logs

• How do langoor logs work?

Langoor logs are a stylish way to log while testing with langoor.
Logs are displayed file-wise.

• Syntax

console.langoor(content);

• Example

test("should log `Langoor rockzzz`!", () => {

 console.langoor(`Langoor rockzzz`);

});
