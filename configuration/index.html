The 21st century javascript unit testing framework.

Configuration

• Where do I put the configuration of my project?

All configurations are stored in a file called langoor.config.js in the root folder of your project.

• Setting up

In your langoor.config.js file, you can ignore files as well as add additional files.

module.exports = {

 additional: [], // Array of additionalfFiles to add to the default ".test.js" files.

 ignore: [], // Ignored files and folder.

};

• Adding files via console

You can add files via console like this:

langoor [file_paths]

For Example:

langoor ./index.js ./client/client.js
